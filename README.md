# Task Management API with Django REST Framework & JWT

## About the project
A RESTful API for managing tasks, built with Django REST Framework and secured using JSON Web Tokens (JWT) for authentication. Users can register, log in, and perform CRUD operations on their tasks. Each userâ€™s tasks are private, accessible only through their authenticated session.

## Features
- User registration and JWT-based login
- Task CRUD operations (create, read, update, delete)
- Token-based authentication for secure access
- Swagger documentation for easy API exploration

## Technologies
- Django
- Django REST Framework
- JWT (Simple JWT)
- drf-yasg for API documentation
